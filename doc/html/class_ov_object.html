Detailed Description

Stripped-down redefinition of OvObject (just for the chunk IDs).

OvObject is a class containing an enum 'Type'. The latter is used for identifying all the possible types of chunks contianed in a .ovo file. Only few of this types are handled in the project, altought a definition for all types is present for better compatibility.
